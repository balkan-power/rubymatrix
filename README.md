# RubyMatrix
Matrix CLI program written in Ruby.

## Prerequisites
You must have Ruby installed on your machine to run this program.
## Usage
### Run by using
> ruby matrix.rb
### Run with different arguments
> ruby matrix.rb -[argument]
### Arguments:
- -c (use half-width kana)
- -C [colour]
- -d [delay]
- -h (help)
- -v (show version)

You can use multiple arguments together to change the appearance of your matrix rainfall.
